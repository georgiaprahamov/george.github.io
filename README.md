# Personal Portfolio Website

A modern, responsive portfolio website built with Flask showcasing my projects and professional experience.

## Features

- Responsive design
- Project showcase
- About me section
- Contact information
- Social media integration
- Downloadable CV

## Tech Stack

- Python (Flask)
- HTML5
- CSS3
- Font Awesome
- JavaScript

## Setup and Installation

1. Clone the repository
```bash
git clone https://github.com/yourusername/portfolio.git
```

2. Create a virtual environment
```bash
python -m venv venv
source venv/bin/activate  # For Mac/Linux
```

3. Install dependencies
```bash
pip install flask
```

4. Run the application
```bash
python main.py
```

5. Visit `http://localhost:5000` in your browser
